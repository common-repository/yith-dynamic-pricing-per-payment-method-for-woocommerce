YITH Dynamic Pricing per Payment Method for WooCommerce allows you to set a discount depending on the payment method used.

== Description ==

YITH Dynamic Pricing per Payment Method for WooCommerce allows setting one or more discounts for each payment method used and applying them to the order total.

= Features: =

* Create one or more discounts based on the payment method selected.

= PREMIUM FEATURES of YITH Dynamic Pricing per Payment Method for WooCommerce: =

* Apply a discount or surcharge on the cart total for each payment method.
* Apply the discount or surcharge as a fixed amount or a percentage.
* Add a short description for each payment rule.
* Apply a discount or surcharge to a specific user role.
* Apply tax to the calculation of each payment method price rule or not
* Set a minimum and/or maximum cart amount required for the payment method rule to apply.
* Schedule start and end date for each rule and make it automatically apply.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Dynamic Pricing per Payment Method for WooCommerce` from Plugins page.

YITH Dynamic Pricing per Payment Method for WooCommerce will add a new submenu called "Dynamic pricing per Payment Method" under "YIT Plugins" menu. Here you are able to configure all the plugin settings.
